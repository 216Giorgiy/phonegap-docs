Connection
==========

> El objeto `connection` permite obtener información del dispositivo móvil y la conectividad wifi.

A este objeto se accede bajo la interfaz navigator.network

Atributos
---------

- connection.type

Constantes
----------

- Connection.UNKNOWN
- Connection.ETHERNET
- Connection.WIFI
- Connection.CELL_2G
- Connection.CELL_3G
- Connection.CELL_4G
- Connection.NONE
